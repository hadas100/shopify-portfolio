אפליקציית יצירת תמונות AI

אפליקציה משולבת ב-Shopify ליצירת תמונות מוצר אוטומטית באמצעות בינה מלאכותית

2024 Leibish Online JavaScript, Node.js, OpenAI API

סקירת הפרויקט

פיתחתי אפליקציה מתוחכמת משולבת ב-Shopify שמנצלת בינה מלאכותית כדי ליצור אוטומטית תמונות מוצר מותאמות אישית. הפתרון מייעל את זרימת העבודה של צילום מוצרים עבור עסקי מסחר אלקטרוני, מפחית משמעותית זמן ועלויות הקשורים לצילום מסורתי של מוצרים.

האתגר

עסקי מסחר אלקטרוני לעיתים קרובות מתקשים ליצור תמונות מוצר עקביות ואיכותיות בקנה מידה גדול. צילום מסורתי גוזל זמן ויקר, במיוחד כשמדובר באלפי וריאציות של מוצרים. תעשיית התכשיטים, במיוחד, דורשת תמונות יפות ומפורטות שמציגות מוצרים מזוויות מרובות ובהגדרות שונות.

הפתרון

בניתי מערכת אוטומטית שמשתלבת בצורה חלקה עם קטלוג המוצרים של Shopify:

  • אחזור מבוסס SKU: האפליקציה שואבת נתוני מוצר ישירות מ-Shopify באמצעות מזהי SKU דרך Shopify REST API
  • יצירת תמונות AI: משתמש ב-APIs של OpenAI DALL-E ו-Google Gemini כדי ליצור תמונות מוצר איכותיות על בסיס מפרטי המוצר
  • העלאה אוטומטית: תמונות שנוצרו מועלות אוטומטית חזרה לרשומות המוצר המתאימות ב-Shopify
  • עיבוד אצווה: תומך ביצירת תמונות בכמות גדולה עבור מוצרים מרובים בו-זמנית
  • בקרת איכות: מיישם בדיקות אימות כדי להבטיח שתמונות שנוצרו עומדות בתקני איכות
