מערכת אינטגרציה ERP-Shopify

סנכרון דו-כיווני מקיף בין Fantasy ERP ו-Shopify

2023-2024 Leibish Online Node.js, GraphQL, REST API

סקירת הפרויקט

פיתחתי מערכת סנכרון דו-כיוונית מקיפה בין Fantasy ERP ו-Shopify, המטפלת בחילופי נתונים בזמן אמת עבור מוצרים, מלאי, הזמנות, מידע על לקוחות ולוגיקה עסקית מותאמת אישית.

האתגר

עסק המסחר האלקטרוני בתכשיטים התמודד עם אתגרים קריטיים עם הזנת נתונים ידנית וסנכרון בין מערכת ה-ERP שלהם לחנות Shopify, מה שהוביל לפערי מלאי ועיכובים בעיבוד הזמנות.

הפתרון

עיצבתי ויישמתי מערכת אינטגרציה חזקה וניתנת להרחבה:

  • סנכרון מוצרים דו-כיווני: סנכרון בזמן אמת של נתוני מוצר כולל כותרות, תיאורים, תמחור ותמונות
  • ניהול מלאי: עדכוני מלאי חיים המבטיחים רמות מלאי מדויקות
  • עיבוד הזמנות: יצירת הזמנות אוטומטית ב-ERP עם עדכוני סטטוס
  • סנכרון נתוני לקוחות: סנכרון חלק של פרופילי לקוחות
  • ניהול Metafields: מיפוי metafield מותאם אישית לנתונים ספציפיים למוצר
  • טיפול בשגיאות ורישום: מעקב שגיאות ומנגנוני ניסיון חוזר מקיפים

ארכיטקטורה טכנית

המערכת משתמשת בארכיטקטורת microservices:

  • API Gateway: ניתוב ואימות מרכזיים
  • מנוע סנכרון: לוגיקת סנכרון ליבה עם עיבוד מונע אירועים
  • ניהול תורים: תורי הודעות לעיבוד אסינכרוני אמין
  • מיפוי נתונים: שכבת טרנספורמציה גמישה

למידה מרכזית
