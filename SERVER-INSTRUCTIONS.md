# הוראות להרצת שרת מקומי

## למה צריך שרת מקומי?
FormSubmit.co דורש שהאתר יפעל דרך שרת HTTP (לא קובץ HTML ישיר). זה נחוץ כדי לבדוק את טופס יצירת הקשר.

## איך להריץ?

### אפשרות 1: שימוש בסקריפט המוכן (הכי קל!)

```bash
./start-server.sh
```

### אפשרות 2: Python ידני

```bash
cd portfolio
python3 -m http.server 8000
```

### אפשרות 3: PHP

```bash
cd portfolio
php -S localhost:8000
```

### אפשרות 4: Node.js (אם יש לך http-server מותקן)

```bash
npx http-server portfolio -p 8000
```

## איך לגשת לאתר?

אחרי הרצת השרת, פתח בדפדפן:
```
http://localhost:8000
```

## בדיקת טופס יצירת הקשר

1. הריצי את השרת (בחרי באחת מהאפשרויות למעלה)
2. פתחי את http://localhost:8000 בדפדפן
3. גלגלי למטה לטופס יצירת הקשר
4. מלאי את הטופס ושלחי
